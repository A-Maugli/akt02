# API calls from Typescript, algosdk version: 3.2.0

Initially:
```
npm init -y
nano package.json
"name": "ts_api"
"description": "TS SDK samples"
CTRL/X
npm add algosdk
npm add typescript --save-dev
npx tsc --init
nano tsconfig.json
"outDir": "./dist"
CTRL/X
npm add @types/node --save-dev
npx tsc
```

After loading from github:
```
npm install
```
